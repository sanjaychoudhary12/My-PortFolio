A modern and responsive **Java Developer Portfolio Website** showcasing skills, projects, and contact information. Built using HTML, CSS, and JavaScript with a clean UI and interactive elements.

## 📌 Features

* 🎯 Responsive design (mobile + desktop)
* 🎨 Modern UI with gradient navbar
* ⚡ Smooth scrolling and animations
* ⌨️ Typing animation (Typed.js)
* 🔄 Interactive skill cards (flip effect)

  * jQuery
  * Typed.js
  * Owl Carousel
  * Font Awesome
